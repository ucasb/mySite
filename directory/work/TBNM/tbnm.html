Thom Browne Window Display

Thom Browne Pop Up - Beverly Hills, CA

One of my first projects, the design is only notable in that I had to replicate an existing 'window', apparently a Thom Browne signature, without access to the original parts (super-fancy, custom aluminum extrusion made in Italy). The copy would need to be cosmetically identical, since they would be placed side-by-side with the originals. The end result turned out so nicely that the windows continue to travel the world to different pop-up shops.
