Laser-Cut Birdhouse

No-tool assembly kit

Finger-jointed birdhouse assembled from 2 laser-cut sheets of 16" x 16" 1/4" ply or similar material. Modeled in Rhino, it's a 3-walled structure with an optional fourth wall that can be slid though the front. Theoretically, an open design is better for birdhouses, while a closed design is better for bird feeders. Pictured example currently hangs on the Chesapeake Bay, with the occasional visitor.

Jeopardy Box

Arduino-based console for playing Jeopardy at home

A COVID project made only with parts available at the makerspace, the box is programmed with traditional Jeopardy rules and features host controls (on the box), 3 color-coded player buzzers linked via TRRS audio cables, a speaker with unique sounds for all actions, and two seperate LED circuits, all housed in a laser cut enclosure and powered by a cheap Arduino Nano clone. I'm particularly proud of the 10-unit Neopixel array used as a countdown indication, which act just like the real contestant podiums! How it works:

  1. In its default state (State A), box awaits host command.
  2. Host reads the clue, and opens the lines with the white button (State B). If the player buzzes in before State B, they are disqualified until the box returns to State A. A ten-second timer begins - if no players buzz in, the box resets to State A.
  3. If a player does buzz in, the box goes into State C, and the bottom LED strip glows either Red, Blue, or Green depending on the player. A 10 second timer begins. If the player answers correctly, the host presses the red reset button, and returns to State A - otherwise, the host presses the white button, and returns to State B, where any remaining qualified players can buzz in.

Signature Ceramics Stamp

3D printed hand stamps + case

Made for my mother-in-law, who sketched her signature design on her iPad and sent it to me. Drawing was vectorized and imported into Solidworks for a simple hand stamp. Printed on a Formlabs Form 3+ using their Tough 1500 resin at a 0.1mm layer height.
